这里要注意，在 base namespace 里面也有一个 TaskRunner 类，SingleThreadTaskRunner 是 base::TaskRunner 的子类，在 base::TaskRunner 中定义了 PostTask 和 PostDelayedTask 方法，其中前者是非虚函数，后者是虚函数，前者会调用后者，而后者交给子类实现，也就是这里的 base::TaskQueueImpl::TaskRunner。
