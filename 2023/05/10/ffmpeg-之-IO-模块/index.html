ffmpeg 之 IO 模块
2023-05-10 21:33:35 # ffmpeg

这篇文章总结一下 ffmpeg 中常见的 IO 函数。

avio_open2

ffio_open_whitelist

这个函数先初始化 URLContext，然后初始化 AVIOContext

ffurl_open_whitelist

ffurl_alloc

先通过 url_find_protocol 查找协议，比如 file、http、ftp。然后创建 URLContext

url_find_protocol
url_alloc_for_protocol

ffurl_connect

这一步会调用 URLProtocol 上的 url_open2 或者 url_open 函数进行连接

比如 file 协议的 open 就是调用系统的 open 来打开文件

ffio_fdopen

在创建 URLContext 后是创建 AVIOContext。URLContext 是不带缓存的，而 AVIOContext 是带缓存的。

avio_alloc_context

ffio_init_context

avio_read

fill_buffer

read_packet_wrapper

read_packet 就是上面 ffio_fdopen 中的 ffurl_read，s->opaque 就是 AVIOContext 关联的 URLContext

ffurl_read

retry_transfer_wrapper

avio_write

avio_flush

flush_buffer

writeout

write_packet 就是 ffurl_write

ffurl_write
